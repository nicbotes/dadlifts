# DADLIFT

Training tracker. Barbell + calisthenics. Built between sets, for the boys.

## Stack
- React + Vite
- Artifact storage for persistence
- No backend

## Dev
```bash
npm install
npm run dev
```

## Deploy
Push to main → Netlify auto-deploys.

## Progression updates
Edit `CALI_CONFIG` in `src/App.jsx` to advance to next skill level.
